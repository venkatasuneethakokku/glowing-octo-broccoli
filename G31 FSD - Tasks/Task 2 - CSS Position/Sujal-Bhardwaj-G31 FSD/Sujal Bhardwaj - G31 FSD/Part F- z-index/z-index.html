Explain How z-index Affects Stacking Order

Definition:
The z-index property controls the stacking order of overlapping elements along the z-axis (front-to-back direction).
Rules:
1. Elements with a higher z-index appear in front of elements with a lower z-index.
2. z-index works on positioned elements (relative, absolute, fixed, or sticky).
3. If two elements have the same z-index, the element that appears later in the HTML code is displayed on top.
