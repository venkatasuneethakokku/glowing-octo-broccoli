CSS Positioning Challenge Questions (28–35)

28. What is the default value of the position property?

Answer: The default value of the position property is static.

29. Which position type removes an element from the normal document flow?

Answer: absolute and fixed positioning remove elements from the normal document flow.

30. Which position type keeps an element visible during scrolling?

Answer: fixed positioning keeps an element visible while scrolling.

31. Which position type requires top, left, right, or bottom values to create a sticky effect?

Answer: sticky positioning requires values such as top:0.

32. What happens when multiple elements have the same z-index value?

If multiple elements have the same z-index value, the element that appears later in the HTML code will appear on top.

33. When would you use absolute positioning instead of relative positioning?

Absolute positioning is used when an element must be placed at an exact location inside a parent container, such as notification badges, tooltips, dropdown menus, or image labels.
